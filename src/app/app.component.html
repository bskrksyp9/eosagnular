This is a demonstration of Token transfer within EOS test accounts.

How does it work ?

- Smart Contract of custom token called SYS has been deployed to Junglenet (Test Network for EOS Blockchain)

- With the help of EOSJS library, the transaction will take place and each record will be registered.

* It takes two test accounts baccount1234 and bktestacc123, with a default value of 1.0000 SYS
